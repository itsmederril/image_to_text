# Image to Text - OCR Web App

Aplikasi web sederhana dan berwarna untuk mengubah gambar menjadi teks menggunakan OCR (Optical Character Recognition).

## Fitur

- **Drag & Drop, Paste, atau Klik untuk Upload Gambar:**  
  Upload gambar dengan mudah melalui drag & drop, paste (Ctrl+V), atau klik area upload.
- **Proses OCR Otomatis:**  
  Teks dari gambar langsung diekstrak secara otomatis.
- **Progress Bar:**  
  Menampilkan progres proses OCR secara real-time.
- **Tombol Copy Text:**  
  Salin hasil teks dengan satu klik.
- **Tampilan Colorful & Modern:**  
  Desain menarik dan nyaman digunakan.

## Cara Penggunaan

1. **Clone atau Download** repository ini "git clone https://github.com/itsmederril/image_to_text".
2. Lalu run > start debugging > selesai.
3. Drag & drop, paste, atau klik area upload untuk memilih gambar.
4. Tunggu proses OCR selesai.
5. Salin hasil teks dengan tombol "Copy Text".
